# Card Spell Generator

A card spell generator on A4 sheets of all DnD 5e spells based on a excel database. Easy to use!


## Authors

- [Gonzalo Bianchini](https://www.github.com/GonzaBianchi)
- [Fernando Lomazzi](https://github.com/FernandoLomazzi)




## Usage/Examples
1. You can create any quantity of spells cards that you want just by adding their spell name on the "spell_to_print.txt"
2. Run main.py to generate "spells_imprimir.csv" which will be used by the scripts.
3. print(SpellCards) and Enjoy them!
